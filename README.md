<h1 align="center"> Sport Stats</h3>

<p>Установить необходимые библиотеки в директорий проекта:</p>

     <p> 1. cd ./SportStats/ </p>
     <p> 2. npm i </p>
     <p> 3. npm i react-router-dom </p>
     <p> 4. npm i axios </p>
     <p> 5. npm i sass </p> 

<p> Запустить веб-приложение: </p>

    <p> npm run dev </p>

<p> Вставить ваш ключ от API монжо в файле: </p>

   <p> ./config.jsx </p>
