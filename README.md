<h1 align="center"> Sport Stats</h3>

<p>Установить необходимые библиотеки в директорий проекта:</p>
     1. cd ./SportStats/
     2. npm i
     3. npm i react-router-dom
     4. npm i axios
     5. npm i sass
Запустить веб-приложение:
    npm run dev
Вставить ваш ключ от API монжо в файле:
    +./config.jsx
